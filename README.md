Project Overview

Glow Beauty Salon is a beauty salon based in Johannesburg, South Africa. The salon offers professional beauty services including hair styling, nail treatments, and makeup services. The purpose of this website is to provide an online platform where customers can learn more about the salon, view services and prices, and contact the business.

The website was developed using HTML, CSS, and JavaScript to create a professional, responsive, and user-friendly online presence for the salon.

⸻

Website Goals and Objectives

The main goal of the website is to improve the salon’s online visibility and attract more customers.

Objectives
	•	Promote beauty services online.
	•	Provide information about the salon.
	•	Display available services and prices.
	•	Allow customers to contact the salon easily.
	•	Increase customer engagement and enquiries.
	•	Create a professional online presence.
